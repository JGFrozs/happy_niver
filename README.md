# 🎉 Happy Niver - Alice & Davi

[PT-BR] Um sistema interativo de celebração desenvolvido com FastAPI, focado em proporcionar uma experiência dinâmica de contagem regressiva e homenagens em vídeo.

[EN] An interactive celebration system built with FastAPI, focused on providing a dynamic countdown experience and video tributes.

---

## 🚀 Tecnologias / Technologies

* **Backend:** Python 3.10+ (FastAPI)
* **Frontend:** Jinja2 Templates, Tailwind CSS
* **Server:** Uvicorn / Gunicorn
* **Deployment:** Render

---

## 🛠️ Funcionalidades / Features

### [PT-BR]
* **Contagem Regressiva:** Sincronizada em tempo real para o momento do aniversário.
* **Mídia Dinâmica:** Suporte para vídeos pesados e alta performance de carregamento via streaming.
* **Lista de Convidados:** Gerenciamento centralizado de participantes.
* **Interface Responsiva:** Otimizado para dispositivos móveis (Android/iOS).

### [EN]
* **Countdown:** Real-time synchronized countdown to the birthday moment.
* **Dynamic Media:** Support for high-quality video tributes with optimized streaming.
* **Guest List:** Centralized management of participants.
* **Responsive UI:** Optimized for mobile devices (Android/iOS).
